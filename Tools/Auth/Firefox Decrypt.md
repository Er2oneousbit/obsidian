#FirefoxDecrypt #crendentals #pillaging 

- Decrypt files in Firefox that may have credentials
- [GitHub - unode/firefox_decrypt: Firefox Decrypt is a tool to extract passwords from Mozilla (Firefox™, Waterfox™, Thunderbird®, SeaMonkey®) profiles](https://github.com/unode/firefox_decrypt)
- `python firefox_decrypt.py`
- `python firefox_decrypt.py /folder/containing/profiles.ini/`
- `python firefox_decrypt.py | grep -C2 keyword`
- `python firefox_decrypt.py --format csv`
- `python firefox_decrypt.py --format pass --pass-username-prefix 'login: '`
